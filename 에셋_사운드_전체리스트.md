# 사운드 에셋 전체 리스트 (BGM / SE 마스터 체크리스트)

> (가제) 오픈챗에서 만나요 · Ren'Py
> **스크립트가 실제로 부르는 오디오 경로를 전수 추출**해 만든 목록입니다.
> (`script_ep1~4`, `epilogue` 의 `pmusic/psound/pamb` + `screens_chat/map` 의 알림음 훅)
> - 적용 코드/세부는 → `제작가이드_그래픽_사운드.md`
> - 모든 오디오는 **에셋이 없으면 자동 무음**(safe-play)이라, 지금 없어도 게임은 정상 진행.

---

## 0. 폴더 · 포맷 · 채널

- 폴더: `game/audio/bgm/` (배경음악), `game/audio/se/` (효과음·앰비언스).
- 포맷: **OGG 권장**(루프·용량 우수). 아주 짧은 효과음은 WAV도 OK.
- 채널:
  - **music** — BGM(루프). `pmusic(path)` / `pstop()`.
  - **sound** — 단발 효과음. `psound(path)` 및 채팅/맵 알림음.
  - **amb** — 앰비언스 루프(거리·카페 소음). `pamb(path)` / `pamb_stop()`.
    - ⚠ 'amb' 채널은 `effects.rpy`에서 **정식 등록**해 둠(이전의 미등록 'audio' 채널 버그 수정).
- 웹 최적화: 개별 50MB↓. BGM은 OGG **96~128kbps**면 충분. 루프 음원은 **이음새 없는 루프**로.

## 1. BGM (배경음악) — 실제 사용 **8곡** (이게 전부)

> 첫 작품은 이 8곡이면 끝. 무드별 1곡씩, 루프 가능하게.

| 파일 (`audio/bgm/…`) | 무드 | 쓰이는 곳 (파일:라인) | 루프 | 길이 권장 |
|---|---|---|---|---|
| `bgm_lonely.ogg` | 적막·외로움 | 프롤로그 도입 (ep1:47) | ✔ | 1~2분 |
| `bgm_daily.ogg` | 잔잔·일상·약간 설렘 | 일상·채팅·카페 다수 (ep1:79·191·415, ep2:33, ep3:36·162, ep4:36) | ✔ | 1~2분 |
| `bgm_comedy.ogg` | 가볍고 통통·텐션 | 코인노래방(ep1:404), 건대 술자리(ep2:115) | ✔ | 1~2분 |
| `bgm_seoa.ogg` | 화사·빠른 설렘 | Ep.1 서아 테마(ep1:320·472) | ✔ | 1~2분 |
| `bgm_warm.ogg` | 따뜻·잔잔·진심 | 진심 씬·지우·엔딩 다수 (ep1:152·395, ep2:207, ep3:113·175·186, ep4:158, epi:17·99) | ✔ | 2~3분 |
| `bgm_night_talk.ogg` | 차분·쓸쓸·밤대화 | 문래 술자리·밤 대화(ep2:101·124, ep4:126) | ✔ | 2~3분 |
| `bgm_tension.ogg` | 묵직·불안 | Ep.4 반전 직전(ep4:90) | ✔ | 1~2분 |
| `bgm_emotional.ogg` | 벅참·해소 | 취중진담·클라이맥스·엔딩 (ep1:115, ep2:187, ep3:225, ep4:207·238, epi:28) | ✔ | 2~3분 |

- **가장 많이 쓰이는 건 `bgm_warm`·`bgm_daily`·`bgm_emotional`** → 퀄리티에 먼저 투자.
- 적용: `pmusic("audio/bgm/bgm_warm.ogg")` (이미 스크립트에 다 연결돼 있음). 파일만 넣으면 바로 들림.

## 2. SE (효과음 / 앰비언스)

### 2-A. 코드에 **이미 연결됨** (파일만 넣으면 바로 작동)

| 파일 (`audio/se/…`) | 용도 | 연결 위치 | 비고 |
|---|---|---|---|
| `se_phone.ogg` | 전화벨(취한 도윤의 새벽 전화) | ep1:116 `psound` | 짧게 1~2회 울림 |
| `amb_street.ogg` | 거리 앰비언스(루프) | ep1:44 `pamb` (amb 채널) | 차 소리·도시 소음, 잔잔한 루프 |
| `se_msg_recv.ogg` | **메신저 '마실' 수신음** | `screens_chat.recv()` | ★새로 연결됨. 받은 메시지마다 |
| `se_msg_send.ogg` | **전송음** | `screens_chat.send()` | ★새로 연결됨. 보낸 메시지마다 |
| `se_station.ogg` | 2호선 역 선택/이동음 | `screens_map` (역 선택 시) | ★새로 연결됨. 역 탭 시 |

> 위 5개는 **safe-play**라 파일이 없으면 그냥 무음. `audio/se/`에 넣는 순간 자동으로 들립니다.

### 2-B. 추가하면 좋은 것 (현재 코드엔 미연결 — 원하면 한 줄로 연결)

| 파일 (`audio/se/…`) | 용도 | 연결 방법(예) |
|---|---|---|
| `se_click.ogg` | UI 버튼 클릭음 | gui에서 `style button: activate_sound "audio/se/se_click.ogg"` |
| `se_typing.ogg` | 입력 중(…) 소리 | `screens_chat.recv()` 의 typing 표시 직후 `psound` 1회 |
| `se_heartbeat.ogg` | 두근거림(고백 직전) | 고백 씬(ep1_confess / ep3_confess) 앞에 `psound` |
| `se_pour.ogg` | 술 따르는 소리 | 문래/건대 술자리(ep2) 분위기용 `psound` |
| `amb_cafe.ogg` | 카페 소음(루프) | 카페 씬에서 `pamb("audio/se/amb_cafe.ogg")`, 나갈 때 `pamb_stop()` |

> 필요하면 위 연결 코드도 제가 넣어드릴 수 있습니다(에셋 준비되면).

## 3. ⚠ 상표·저작권 경고 (중요)

- **실제 메신저(카카오톡 등)의 알림음을 절대 복제·사용 금지.** 그 사운드는 보호 대상.
  → '마실' 수신/전송음은 **직접 만들거나 CC0 효과음**으로 완전히 다른 소리를 쓸 것.
- 매장 BGM이 녹음된 **현장음(거리/카페 녹음)** 도 피하기(타 음원 저작권 혼입 위험). 앰비언스는 CC0 환경음으로.

## 4. 무료/안전 소스 (라이선스 항목별 직접 재확인)

- **공유마당(한국저작권위원회)** — 한국형 효과음·음원, 상업 이용 가능(대개 출처 표기 조건).
- **Freesound** — 효과음 대량, **CC0 필터**로 무출처 사용.
- **BigSoundBank / Kenney / OpenGameArt** — CC0 효과음·음악.
- **itch.io** — VN용 BGM/SFX 팩(상업 가능·CC0 태그 확인).

> 사용한 음원·라이선스·출처는 `CREDITS.txt`에 기록(CC-BY는 크레딧 필수).

## 5. 우선순위 (첫 작품 기준)

1. **BGM 3곡 먼저**: `bgm_daily` · `bgm_warm` · `bgm_emotional` (가장 자주 쓰임).
2. **채팅 알림음 2개**: `se_msg_recv` · `se_msg_send` (게임 정체성 = 메신저, 체감 큼).
3. 나머지 BGM 5곡 → `amb_street` · `se_phone` → `se_station`.
4. 여력껏 2-B(클릭/두근/카페 앰비언스)로 디테일.
