> [!IMPORTANT]
> **이 게임은 “여러 여자와 썸타는 게임”이 아닙니다.**
> 연애에 서툴고 한 번 데인 20대 남자가, 오픈챗에서 만난 진짜 친구 **도윤**을 통해 ‘진심’을 배워가는 이야기입니다.
> 연애는 **소재**, 우정과 진심은 **주제**.

원래 Ren'Py로 만들던 작품을 **빌드 없는 순수 웹앱**으로 옮겼습니다. 정적 파일이라 Vercel에 폴더째 올리면 끝 — 무거운 엔진/헤더 없이 PC·모바일 웹 어디서나 돌아갑니다.

## ✨ 특징

| | |
|---|---|
| 💬 **마실 채팅 UI** | 카톡류를 베끼지 않은 독자 디자인 메신저. 말풍선·타임스탬프·‘읽음’·입력중 애니메이션·프사 이니셜. 반응형. |
| ⏱ **답장 타이밍 / 읽씹** | 상대 톡에 `바로 / 뜸 들이고 / 못 본 척`. **상대마다 다르게 읽혀** 정답이 하나가 아님. |
| ❤️ **호감 ≠ 진심 (비표시 이중 수치)** | “잘 보이긴 쉽고 진심은 비싸다”를 시스템으로. **숫자는 일부러 숨김** — 도윤의 말과 친구목록의 ‘결’로 간접 표현. |
| 🧑‍🤝‍🧑 **도윤 — 주인공급 동생** | 모든 화를 관통하는 상수. 상담으로 힌트를 주고, 우정이 깊어질수록 조언이 정확해짐. |
| 🚇 **2호선 맵** | 거점 잠실에서 진행에 따라 역이 해금. |
| 🎁 **아이템 / 추억함** | 벚꽃 엽서·도윤의 키링·폴라로이드… 건넨 물건이 분기와 정서를 만든다. |
| 🏆 **엔딩 7종 + 진단** | 진/굿/어장/도윤 우정/화해 진/쓸쓸/도피 + 엔딩 후 “당신은 ___형” 연애유형 진단 카드. |

## 🗺 진행 구조

```
[프롤로그]  오픈챗 첫 입장 · 주인공의 결함(외로움·데인 과거) · 도윤과 ‘그 술자리 밤’
[Ep.1] 서아  — 빠르게 불타는 썸        (신촌)
[Ep.2] 도윤  — 문래, 그 골목의 밤      (우정 심화)
[Ep.3] 지우  — 천천히 진짜가 되는      (성수 슬로우번)
[Ep.4] 민결  — 돌아온 이름            (4지선다 결말)
[에필로그]  누적된 진심·우정에 따라 7엔딩 분기 + 연애 유형 진단
```

시간순 옴니버스 + 진심·우정 누적. 매 화 끝 도윤의 코멘트로 누적 감각을 줍니다.

## 🚀 실행하기

빌드 과정이 없습니다. 정적 서버만 있으면 됩니다.

```bash
cd web
python3 -m http.server 8000
# → http://localhost:8000
```

> 게임 내 우상단 **📱 버튼**으로 친구목록 / 추억함 / 갤러리에 접근합니다. *(메타 화면은 다음 마일스톤)*

### 🌐 Vercel 배포

1. [vercel.com](https://vercel.com) → **Add New → Project** → 이 저장소 import
2. **Root Directory** 를 **`web`** 으로 지정 *(프로젝트 루트가 아니라 `web/` 폴더)*
3. Framework Preset **Other**, 빌드 설정 없음 → **Deploy**

`web/vercel.json` 포함 (정적 서빙, COOP/COEP 불필요). 한글 폰트(Pretendard)는 내장돼 있어 어느 기기에서도 안 깨집니다.

## 🧱 프로젝트 구조

배포되는 건 **`web/` 폴더 하나**입니다.

```
web/
├─ index.html              진입점 (ES 모듈 로드)
├─ style.css               반응형 레이아웃 + 마실 테마 + @font-face
├─ vercel.json             정적 배포 설정
├─ favicon.ico / icon-256.png
├─ fonts/                  Pretendard · Jua (자체 호스팅, SIL OFL)
├─ audio/
│  ├─ se/                  효과음 SE (CC0 Kenney) + 앰비언스 · BGM은 audio/bgm/ 에
│  └─ CREDITS.txt          사운드 라이선스/출처
├─ data/
│  ├─ story.json           전체 스토리 (730노드 · 40라벨) — 콘텐츠 소스
│  └─ characters.json      캐릭터 정의(이름·색)
└─ src/
   ├─ engine.js            노드 스크립트를 실행하는 상태머신
   ├─ state.js             게임 상태 + localStorage 세이브
   ├─ systems.js           게이지·아이템·엔딩 판정·도윤 상담·답장 타이밍
   ├─ eval_expr.js         조건/식 평가 (V/S/P 스코프)
   ├─ theme.js             테마·아이템·엔딩 상수
   ├─ settings.js          설정(음악·사운드·밝기·진동) localStorage 저장
   ├─ audio.js             효과음·BGM·진동(햅틱) 매니저 (설정 연동)
   ├─ playback.js          스킵·오토 모드 + 대사 히스토리 + 롤백 스냅샷
   └─ ui/                  stage · chat · menu · overlay · map · phone
                           · settings_ui · title · saveload · sysmenu · backlog · view_dom
```

- **엔진**: `data/story.json` 의 노드 배열을 위에서 아래로 실행, 입력 노드(대사·선택·입력)에서 멈춤. 분기는 라벨/점프.
- **저장**: 전부 `localStorage`. 서버·DB 없음.
- **세이프 플레이**: 배경/사운드/프사 에셋이 없어도 무음·단색·이니셜로 통과(현재 배경은 단색 placeholder).

## 🛠 기술

`바닐라 JS(ES 모듈)` · `HTML/CSS` · `localStorage` · **빌드 단계 없음** · 외부 런타임 의존성 0.
테스트는 Node 내장 러너: `node --test test/*.test.js` (엔진 + 전체 스토리 플레이스루).

## 🎮 게임 기능 (현재 구현)

플레이로 체감되는, 지금 실제로 동작하는 기능들:

- **마실 채팅** — 말풍선·타임스탬프·‘읽음’·입력중(…) 애니메이션·프사 이니셜, 수신/전송 효과음
- **답장 타이밍 / 읽씹** — `바로 / 뜸 / 못 본 척`, 상대마다 다르게 읽힘(정답이 하나가 아님)
- **호감 ≠ 진심 이중 게이지(비표시)** — 숫자 숨김, 피드백은 도윤의 말·친구목록의 ‘결’ 같은 비수치로
- **도윤 상담** — 챕터당 1회, 우정이 깊을수록 조언이 정확
- **아이템 / 추억함** — 벚꽃 엽서·키링·폴라로이드 등 건네고 간직 → 분기·정서
- **2호선 맵** — 진행에 따라 역 해금, 현위치 ♥ 표시
- **친구목록 / 갤러리 / 진단** — 비수치 관계 ‘결’ 라벨, 엔딩 수집 갤러리, 엔딩 후 연애유형 진단
- **7가지 엔딩** — 명시적 선택(Ep.4)·우정·게이지로 분기
- **타이틀/메인메뉴** — 새로하기 · 이어하기(오토세이브 복원) · 불러오기 · 설정
- **세이브 / 불러오기** — 슬롯 6개 + 퀵세이브, 각 슬롯 미리보기(챕터·화자·대사·시간). 전부 `localStorage`
- **퀵세이브·퀵로드** — `F5` / `F9` (또는 시스템 메뉴 버튼)
- **시스템 메뉴(게임 중)** — 스킵(읽기 빨리감기) · 오토(자동 진행) · 백로그(지난 대사) · 롤백(이전으로) · 저장/불러오기 · 메뉴로
- **효과음(SE) + 앰비언스** — 채팅·클릭·역·아이템·전화 등 (CC0 Kenney, [크레딧](web/audio/CREDITS.txt)) · BGM은 제작 예정
- **모바일 진동(햅틱)** — 메시지 수신·아이템·전화·엔딩 등 주요 순간 (설정으로 on/off)
- **설정 메뉴(⚙️)** — 음악·사운드 볼륨, 밝기, 진동 (localStorage 저장)
- **반응형 + 한글 폰트 내장(Pretendard·Jua)** — PC/모바일, 파스텔 코지 UI
