# 《오픈챗에서 만나요》 — 전체 분기 공략 (Behind-the-scenes Walkthrough)

> ⚠️ **이 문서는 제작/공략용**입니다. 게임 본편은 이 수치들을 **플레이어에게 절대 노출하지 않습니다**(피드백은 도윤의 말·친구목록의 '결' 같은 비수치로만). 스포일러 포함.
>
> 구조: **Ep.1 서아 → Ep.2 도윤 → Ep.3 지우 → Ep.4 민결 → 에필로그(엔딩)**. 매 화 [텍스팅 인터루드 → 도윤 콜백 → 만남 → 장소 분기 → 결말 선택 → wrap]. 엔딩은 **7종**.

## 1. 숨은 시스템 (게이지·플래그·아이템)

### 게이지 (비노출)
| 게이지 | 의미 | 범위 |
|---|---|---|
| `like[히로인]` (서아/지우/민결) | 호감 (잘 보이는 것) | 0~100 (clamp) |
| `sincere[히로인]` (서아/지우/민결) | **진심** (솔직·취약·깊이) — 엔딩의 핵심 | 0~100 (clamp) |
| `doyun_bond` | 도윤과의 우정·신뢰 | **누적(상한 없음)** — Ep.2에서 크게 오름 |

- **핵심 원칙: 진심(sincere) > 호감(like).** 호감만 높고 진심이 낮으면 '어장' 취급(fishtank 엔딩). 진엔딩·트루엔딩은 전부 **sincere 임계**로 갈림.
- 답장 타이밍(`apply_timing`): 상대마다 다름.
  - **서아**: 바로=호감+10 / 뜸=호감+3 / 무시=호감 **-5**
  - **지우**: 바로=호감+5 / 뜸=**진심+8** / 무시=호감+2  ← 지우는 '뜸 들이는 신중함'을 진심으로 침
  - **민결**: 바로=호감+3 / 뜸=호감+8 / 무시=**진심+4**  ← 민결은 거리감을 편해함

### 주요 플래그
| 플래그 | 켜지는 곳 | 영향 |
|---|---|---|
| `honest_doyun` | Ep.1 도윤 인터루드 '솔직히 인정' | Ep.1 wrap 도윤 콜백 변주 |
| `noticed_doyun` | Ep.2 '깨면 또 혼자' 짚어주기 | Ep.2 다음날 아침 도윤 콜백 변주 |
| `leaned_jiu` | Ep.3 지우에게 '한 발 내딛기'('꼭 그것만은 아니어도') | Ep.3 wrap 도윤 콜백 변주 |
| `doyun_secret_seen` | Ep.2 '왜 그렇게 무너졌는지 묻기' | Ep.4 민결 진실 프레이밍 |
| `seoa_result` | Ep.1 결말 (breakup/fizzle/slow) | Ep.2 도입 도윤 톡 변주 |
| `promise_spring` | Ep.1 벚꽃 엽서 만들기 | 에필로그 마무리 변주 |
| `ep4_choice` | Ep.4 최종 선택 | **엔딩 1차 분기** |

## 2. 엔딩 7종 + 도달 조건 (`final_ending()` 결정 트리)

엔딩은 **Ep.4 최종 선택(`ep4_choice`)을 먼저 보고**, 그 조건을 충족하면 확정 / 못 채우면 **전체 진심 게이지 기반(`decide_ending`)으로 fall-through**합니다.

```
1) ep4_choice == "run"                                          → ⬛ run (다시 혼자)
2) ep4_choice == "reconcile" && doyun_bond≥25 && sincere.민결≥35 → 🌟 reconcile (용서까지 데려다준 사람) [진엔딩]
3) ep4_choice == "friend"   && (doyun_bond≥25 || 키링 보유)      → 💚 doyun (그날 그 손을 끝까지)
4) ep4_choice == "love"     && sincere.민결≥40                  → 💖 true (끝내 건넨 진심 · 민결)
5) 위 조건 미충족 → decide_ending():
     · 히로인 2명↑이 like≥70 & sincere<30                       → 🐟 fishtank (모두의, 아무도 아닌)
     · 진심 최고 히로인 best: sincere[best]≥70 & like[best]≥60   → 💖 true (끝내 건넨 진심 · best)
     · sincere[best]≥50                                          → 🧡 good (서툰 진심 · best)
     · 그 외                                                     → 🤍 lonely (못다 준 사람)
```

### 엔딩별 요약
| 엔딩 | 이름 | 한 줄 | 핵심 조건 |
|---|---|---|---|
| 🌟 **reconcile** | 용서까지 데려다준 사람 | **진엔딩** — 도윤도 민결도 잃지 않은 최난도 | Ep.4 '셋이 풀어본다' + `doyun_bond≥25` + `sincere.민결≥35` |
| 💚 **doyun** | 그날 그 손을 끝까지 | 사랑보다 곁(도윤)을 택함 | Ep.4 '의리를 지킨다' + (`doyun_bond≥25` 또는 **키링**) |
| 💖 **true** | 끝내 건넨 진심 | 비싼 진심을 한 사람에게 | Ep.4 '민결 선택'+`sincere.민결≥40`, **또는** 누군가 `sincere≥70 & like≥60` |
| 🧡 **good** | 서툰 진심 | 완벽친 않아도 도망 안 침 | 진심 최고 히로인 `sincere≥50` (진엔딩 미달) |
| 🐟 **fishtank** | 모두의, 아무도 아닌 | 잘 보이기만 한 어장 | 히로인 2명↑ `like≥70 & sincere<30` |
| 🤍 **lonely** | 못다 준 사람 | 진심은 늘 한 발 늦게 | 어느 히로인도 진심 임계 못 넘음 |
| ⬛ **run** | 다시 혼자 | 가장 익숙하고 쓸쓸한 | Ep.4 '도망친다' |

> **함정**: Ep.4에서 reconcile/love/friend를 골라도 **임계 미달이면 그 엔딩이 아니라** 5)의 진심 기반 엔딩으로 떨어집니다. 예) reconcile 골랐는데 doyun_bond 24 → 진엔딩 X, decide_ending 결과(good/lonely 등).

## 3. 화별 선택지 효과표

### Ep.1 — 서아 (빠르게 불타는 썸)

| | 가볍게 | 호감 +5 |
| 첫 인사(데이트) | 농담으로 | 호감 +15 |
| | **떨린다고 솔직히** | **진심 +15** |

| | **솔직히 인정** | **진심 +15** |

## 4. 엔딩별 추천 루트 (요약)

> 모든 회차에서 **답장은 상대 성향대로**(서아·민결은 상황별, 지우는 '뜸'), 진심 선택 위주.

- 🌟 **진엔딩 reconcile** (가장 어려움)
  - **Ep.2에서 도윤 bond를 25 이상으로** 확실히(짚기·문래·솔직·숙취제·친구니까·다털기).
  - **Ep.4 민결 sincere를 35 이상으로**(무시 답장 +진심, 물러서 인정, 벽을 이해 +15 → 합 ~29+, 도윤이야기까지면 충분). 도윤(문래) 먼저 찾아가 키링 보유면 bond 여유.
  - **Ep.4 최종 = '셋이 풀어본다'.**

- 💚 **doyun (의리)**: Ep.2 bond 쌓거나 **키링만 있어도 OK** → Ep.4 최종 '도윤 의리'. (가장 안정적인 우정 엔딩.)

- 💖 **true (민결과 진심)**: Ep.4 민결 sincere **40 이상**(타이밍·물러서·벽이해·도윤이야기 전부 진심) + 최종 '민결 선택'.
  - *또는* 특정 히로인(서아/지우/민결) 하나에 **sincere≥70 & like≥60** 몰아주면 그 사람으로 true.

- 🧡 **good**: 한 히로인 진심을 **50 이상**까지(진엔딩 임계엔 못 미침).

- 🐟 **fishtank**: 여러 히로인에게 **호감만**(가볍게/농담/빠른수락) 몰고 진심은 안 줌 → 2명↑ like≥70 & sincere<30.

- 🤍 **lonely**: 진심을 어디에도 충분히 안 줌(전반적으로 호감·진심 낮게).

- ⬛ **run**: Ep.4 최종 '도망친다'.
