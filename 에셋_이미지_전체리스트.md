## 0. 기준 · 폴더 · 포맷

- **★배경은 PC/모바일 2버전** — PC(가로) = **16:9**, 모바일(세로) = **9:16**. 게임이 기기에 맞춰 자동으로 골라 불러옴. (캐릭터 CG·스프라이트·프사는 1버전, 분리 안 함.)
- **해상도**: 배경 PC 16:9(1920×1080 또는 1280×720) · 배경 모바일 9:16(1080×1920 또는 720×1280) · CG 16:9.
- **포맷**: 배경/CG = `PNG` 또는 `JPG`(배경은 JPG로 용량↓ OK), 스프라이트/프사 = **투명 PNG**.
- **★파일명 = 엔진 키** (`bg_<코드>.png`). 아래 표의 `코드: bg X` 가 키 → 파일은 `bg_X.png`. **게임 플레이 중 좌상단 ‘에셋 힌트’ 칩이 각 장면에 필요한 정확한 경로를 알려줌**(설정에서 끌 수 있음).
- **폴더**(`web/` 아래):
```
web/images/
 ├─ bg/
 │   ├─ pc/      배경 PC(16:9)   (bg_room.png, bg_sinchon.png ...)
 │   └─ mobile/  배경 모바일(9:16) (bg_room.png, bg_sinchon.png ...)  ← 같은 파일명, 세로 구도
 ├─ sprite/    캐릭터 스프라이트(투명) (doyun_base.png ...)
 ├─ cg/        이벤트 CG(16:9)  (cg_doyun_night.png ...)
 ├─ avatar/    채팅 프사(1:1)  (avatar_doyun.png ...)
 └─ ui/        타이틀/로딩 등 (title-logo.svg ...)
```
- **현재 코드 상태**: 모든 배경이 단색 플레이스홀더라 **이미지 없이도 게임은 돌아감.** `web/images/bg/pc/` · `web/images/bg/mobile/` 에 채우면 자동 교체.
- **웹 최적화**: 개별 파일 작게(배경 JPG 품질 80~85%), PC 가로 1280~1920·모바일 세로 720~1080 안에서.

## 1. 배경 (Backgrounds) — ★직접 촬영용 가이드 포함

> 촬영 공통 수칙
> - **★PC(16:9 가로) + 모바일(9:16 세로) 둘 다 준비.** 가로로 찍고 세로는 크롭/확장(같은 장소·톤 유지)해도 됨. 파일명은 같게 두고 `bg/pc/`·`bg/mobile/` 폴더만 다르게.
> - 인물 없이(또는 군중은 멀리/흐리게). **타인 얼굴·실존 간판/로고는 가리거나 피한다.**
> - **화면 하단 1/4은 대사창에 가려짐** → 중요한 디테일을 위/중앙에 두고 아래는 비워서 찍기. (세로 버전은 대사창이 더 크니 더 위로.)
> - 같은 장소의 **낮/노을/밤은 한 번에 여러 시간대로** 찍어두면 색보정만으로 재활용.
> - 촬영 후 처리: 일러스트 필터(페인팅/유화) 또는 img2img로 "한국 웹툰 배경" 톤 통일(스타일은 프롬프트집 STYLE 참고).

> 표정은 **레이어 분리**(몸 1 + 표정 교체) 권장 → 용량·작업 절약. (LayeredImage, 가이드 2-3 참고)
