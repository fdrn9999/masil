# vn-assets — 에셋 감사 (실사용 vs 정의 vs 데드)

예시 말고 **코드가 진짜 부르는 것** 기준으로 에셋 작업 목록을 만든다.

## 절차
1. VN 폴더(`script_ep1.rpy` 있는 곳)로 이동.
2. 아래를 추출·대조한다.

### BGM / SE / 앰비언스 (오디오)
- 실사용 경로: `grep -ron 'audio/(bgm|se)/[A-Za-z0-9_]+\.ogg' *.rpy` → 유니크 목록.
- 각 파일이 **쓰이는 위치(파일:라인)** 와 함께 표로. BGM은 `pmusic`, 단발은 `psound`, 앰비언스는 `pamb`(채널 `amb`).
- 채팅/맵 알림음은 코드에 훅이 있다: `se_msg_recv`, `se_msg_send`(screens_chat), `se_station`(screens_map), `se_phone`(ep1), `amb_street`(ep1). 파일만 넣으면 자동 재생.

### 배경 (image bg)
- 정의: `grep -n '^image\s\+bg ' *.rpy` (이름·색).
- 실사용: `grep -n '^\s*scene bg ' *.rpy`.
- **대조**: 정의됐지만 scene에 안 쓰이면 **데드(안 만들어도 됨)**, scene에 쓰였는데 정의 없으면 **누락(에러 위험)**.
- 중복 색/동의 배경(예: `street_n`=`street_night`, `cafe`=`cafe_in`, `han_night`↔`han_river`)은 **한 장으로 겸용** 가능하다고 표시.

### 채팅 프사 (avatar)
- 자동 연결 규칙: `screens_chat.rpy` 의 `AVATAR_FILES` → `images/avatar/avatar_doyun.png` 등. 파일 있으면 표시, 없으면 이니셜 동그라미.
- 어떤 프사 파일이 기대되는지 그 dict에서 그대로 뽑아 목록화.

### 스프라이트 / CG (현재 미사용)
- 본문은 아직 `Solid` placeholder + 1인칭이라 스프라이트/CG 호출은 거의 없음 → "권장 목록"은 코드가 아니라 **표정/CG가 필요한 감정 비트**(예: 도윤 취함/울컥, 옥상 민결)에서 도출. 에셋 리스트 문서를 따른다.

## 문서 대조
다음과 **상호 검증**하고 불일치를 지적:
- `에셋_이미지_전체리스트.md`, `에셋_사운드_전체리스트.md`, `이미지생성_프롬프트집.md`
- 코드는 부르는데 리스트에 없거나, 리스트에 있는데 코드가 안 부르면 표시.

## 출력
- **만들어야 할 것**(누락) / **있으면 자동 적용**(훅 연결됨) / **데드(스킵)** 3분류로 정리.
- 배경은 "직접 촬영" 항목이면 촬영 메모(장소·시간대·하단 1/4 비우기·실간판 금지)도 함께.
