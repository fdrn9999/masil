> (가제) 오픈챗에서 만나요 · Ren'Py
> 그래픽·사운드 에셋 제작/수급 가이드 + Ren'Py 적용 코드
> ※ 저작권·상표 안전을 최우선으로 정리. 모든 외부 에셋은 라이선스 직접 재확인 필수.

## 0. 해상도·폴더 기준 (먼저 정하고 시작)

- **기준 해상도: 1920 × 1080 (16:9)** 추천. Ren'Py `gui.init(1920, 1080)`.
  - 가벼운 첫 작품이면 1280 × 720 도 무방(용량·작업량↓). 단, **프로젝트 내내 한 비율로 통일**.
- 폴더 구조:

```
game/
 ├─ images/
 │   ├─ bg/        배경 (bg_jamsil_room.png ...)
 │   ├─ sprite/    스탠딩 CG (doyun_normal.png ...)
 │   └─ cg/        이벤트 일러스트(전체화면 한 컷)
 ├─ audio/
 │   ├─ bgm/       배경음악 (.ogg)
 │   └─ se/        효과음 (.ogg / .wav)
 └─ *.rpy
```

- 권장 포맷: 이미지 **PNG**(스프라이트는 투명 알파), 오디오 **OGG**(용량·루프 우수). 효과음 짧은 건 WAV도 OK.

## 1. 배경 일러스트 (Background)

### 1-1. 필요 배경 목록 (현재 스크립트 기준)

| 파일명(예시) | 장소 | 시간대 | 쓰이는 곳 |
|---|---|---|---|
| `bg_jamsil_room` | 잠실 자취방 | 밤 | 프롤로그·거점·일상 |
| `bg_game_screen` | 게임 화면/방 | 밤 | 도윤과 게임방 |
| `bg_sinchon_st` | 신촌 연세로 | 낮 | Ep.1 서아 |
| `bg_cafe` | 카페 내부 | 낮 | Ep.1 카페 |
| `bg_mullae_bar` | 문래 골목 술집 | 밤 | Ep.2 도윤 |
| `bg_street_night` | 밤거리 | 밤 | Ep.2 귀가 |
| `bg_boardcafe` | 보드게임 카페 | 낮 | Ep.3 지우 |
| `bg_seongsu` | 성수 카페거리 | 낮/노을 | Ep.3 데이트 |
| `bg_hangang` | 한강 | 노을/밤 | Ep.3 고백 |
| `bg_gangnam` | 강남 네온 거리 | 밤 | Ep.4 민결 |
| `bg_rooftop_bar` | 옥상 바 | 밤 | Ep.4 만남 |
| `bg_doyun_room` | 도윤의 방 | 낮 | Ep.4 결말 |

> 팁: **같은 장소 + 시간대 변형**(낮/노을/밤)은 색보정만으로 재활용. 노을·밤 버전은 원본에 오버레이 한 장이면 충분.

### 1-2. 스타일 가이드

- 톤: "가벼운 코미디 표면 + 현실 드라마". → **실사 배경 같은 세미리얼 + 부드러운 채도**가 잘 맞음. (쓰르라미·두근두근 문예부류의 사진 가공 배경 톤)
- 실제 장소(홍대·신촌·성수) 분위기를 살리되, **실존 간판·로고·상호는 그리지 말 것**(상표/저작권). 가상의 간판으로 대체.
- 인물 없는 빈 배경으로(스프라이트가 위에 올라감). 화면 하단 1/4은 대사창에 가려지니 중요한 디테일 배치 금지.

### 1-3. 수급 방법 (택1·혼합)

1. **사진 가공**: 직접 찍은 홍대/신촌 사진 → 일러스트 필터/페인팅 가공. 가장 저렴하고 "실제 장소" 컨셉에 딱.
   - 주의: 사진에 **타인 얼굴, 상표 간판** 들어가면 가리거나 변형.
2. **AI 생성**: 배경은 AI로 빠르게 뽑기 좋음. 단 **상업 이용 가능한 툴/플랜인지 약관 확인**(생성물 권리·재배포 조건이 툴마다 다름). 인물 없는 배경 위주로.
3. **유료 에셋/외주**: itch.io·BOOTH의 VN 배경팩, 또는 일러스트레이터 커미션. 라이선스에 "상업적 이용·게임 내장 허용" 명시 확인.

## 2. 스탠딩 CG (캐릭터 스프라이트)

### 2-1. 캐릭터 & 표정 목록

| 캐릭터 | 색(이름표) | 필수 표정(최소) |
|---|---|---|
| 도윤(조력자) | 초록 | 평소(능청), 웃음, 진지, 취함, 울컥 |
| 서아 | 빨강 | 밝음, 새침, 시무룩 |
| 지우 | 파랑 | 잔잔미소, 부끄럼, 쓸쓸 |
| 민결 | 보라 | 날카로움, 흔들림, 진심 |
| (주인공) | — | 보통 스프라이트 없음(1인칭). 필요시 뒷모습/실루엣 |

> 표정은 처음엔 **3~5종이면 충분**. 부족하면 나중에 추가. 감정 폭이 큰 도윤·민결만 표정 많이.

### 2-2. 스펙

- 전신 또는 무릎 위. 캔버스 세로 ≈ 화면 높이의 90% (1080 기준 약 950~1000px), **투명 배경 PNG**.
- **레이어 분리 권장**: 몸(베이스) + 표정(눈/입)만 교체 → 표정 수 늘려도 용량·작업 절약. Ren'Py `LayeredImage`로 구현.
- 발 끝이 대사창에 약간 가려지는 위치를 기본으로.

### 2-3. Ren'Py 적용 — LayeredImage (표정 교체형)

```renpy
# images.rpy 등에 정의
layeredimage doyun:
    # 베이스(몸)
    always "sprite/doyun_base.png"
    # 표정 그룹
    group face:
        attribute normal default "sprite/doyun_face_normal.png"
        attribute smile        "sprite/doyun_face_smile.png"
        attribute serious      "sprite/doyun_face_serious.png"
        attribute drunk        "sprite/doyun_face_drunk.png"
        attribute cry          "sprite/doyun_face_cry.png"

# 스크립트에서 사용
show doyun serious at center with dissolve
d "형. 그 사람… 민결이야."
show doyun cry          # 표정만 교체
```

단순하게 통짜 PNG로 갈 거면:

```renpy
image doyun normal  = "sprite/doyun_normal.png"
image doyun smile   = "sprite/doyun_smile.png"
show doyun smile at center with dissolve
```

위치 프리셋(`left/center/right`)은 Ren'Py 기본 제공. 2인 대화는 `at left` / `at right`.

| 트랙(예시 파일) | 무드 | 쓰이는 곳 |

| `bgm_night_talk` | 차분·쓸쓸 | 문래 술자리, 밤 대화 |

> 첫 작품은 **6~8곡이면 충분**. 무드별로 한 곡씩, 루프 가능한 음원으로.

### 3-2. Ren'Py 적용

```renpy
# 채널은 music 기본 제공, 루프 자동
play music "audio/bgm/bgm_daily.ogg" fadein 1.0
# 씬 전환 시
play music "audio/bgm/bgm_tension.ogg" fadeout 1.0 fadein 1.0
stop music fadeout 2.0   # 엔딩 전 정적
```

스크립트 예: Ep.4 반전 직전에 `play music "audio/bgm/bgm_tension.ogg" fadein 1.0` 한 줄 추가.

## 4. SE (효과음)

### 4-1. 필요 효과음

| 파일(예시) | 용도 |
|---|---|
| `se_msg_recv` | 메신저 '마실' 수신음 (받은 메시지) |
| `se_msg_send` | 전송음 |
| `se_typing` | 입력 중 (선택) |
| `se_click` | UI 버튼 클릭 |
| `se_station` | 맵에서 역 선택/이동 |
| `se_pour` | 술 따르는 소리(문래 술자리) |
| `se_heartbeat` | 두근거림(고백 씬) |
| `amb_street` | 거리 앰비언스(루프) |
| `amb_cafe` | 카페 소음(루프) |

### 4-2. ⚠ 상표·저작권 경고 (중요)

- **실제 메신저(카카오톡 등)의 알림음을 절대 복제/사용 금지.** 그 사운드는 보호 대상. '마실' 수신음은 **직접 만들거나 CC0 효과음**으로 완전히 다른 소리를 쓸 것.
- 실존 상호·음악이 녹음된 현장음(매장 BGM 흘러나오는 거리 녹음 등)도 피할 것.

### 4-3. Ren'Py 적용 (+ 채팅 알림음 연동)

```renpy
# 단발 효과음
play sound "audio/se/se_click.ogg"

# 앰비언스(루프) - audio 채널 말고 별도 채널 권장
play audio "audio/se/amb_cafe.ogg"   # 또는 커스텀 채널
```

screens_chat.rpy 의 `recv()` / `send()` 에 수신·전송음을 끼우려면:

```python
def recv(text, name=None, avatar=None, type_time=0.8):
    chat_log.append({"side":"left","typing":True,"name":name,"avatar":avatar})
    _refresh(); renpy.pause(type_time, hard=True); chat_log.pop()
    chat_log.append({"side":"left","text":text,"name":name,"avatar":avatar})
    renpy.sound.play("audio/se/se_msg_recv.ogg")   # ← 수신음
    _refresh()

def send(text):
    chat_log.append({"side":"right","text":text})
    renpy.sound.play("audio/se/se_msg_send.ogg")    # ← 전송음
    _refresh()
```

버튼 클릭음은 gui 에서 일괄 지정도 가능:

```renpy
define config.main_menu_music = "audio/bgm/bgm_daily.ogg"
style button:
    activate_sound "audio/se/se_click.ogg"
```

## 5. 무료/저작권 안전 리소스 (라이선스는 반드시 직접 재확인)

### 사운드(BGM·SE)
- **공유마당 (한국저작권위원회)** — 한국적 효과음·음원, 상업 이용 가능(대부분 출처 표기 조건). 한국형 효과 음원 라이브러리 있음.
- **Freesound** — 50만+ 효과음, **CC0 필터**로 무출처 사용 가능.
- **Kenney** — 게임 에셋·SFX, 대부분 CC0.
- **OpenGameArt** — 음악·SFX, CC0/CC-BY 등.
- **BigSoundBank** — CC0 1.0 효과음.
- **itch.io** — VN용 BGM/SFX 팩, CC0 태그로 필터.

### 이미지/배경·스프라이트
- **itch.io / BOOTH** — VN 배경·스프라이트 에셋(상업 가능 여부 개별 확인).
- 직접 촬영 사진 가공, 또는 상업 약관이 명확한 AI 툴.

> 공통 원칙: 다운로드 전 **"상업적 이용 가능"** 과 **"출처 표기 의무"** 를 항목별로 확인하고, 사용한 에셋·라이선스·출처를 별도 `CREDITS.txt` 에 기록해 둘 것. (CC-BY는 크레딧 필수)

## 6. 작업 우선순위 (첫 작품 기준)

1. **임시 에셋으로 끝까지 굴리기** — 지금처럼 단색 배경 + 표정 없는 스프라이트로 프롤로그~Ep.4 흐름 완성.
2. 배경부터 채우기(장소 분위기가 체감을 가장 크게 바꿈).
3. 스프라이트(도윤·서아·지우·민결, 표정 3~5종).
4. BGM 6~8곡 → SE.
5. 마지막에 이벤트 CG(고백·엔딩 한 컷씩)로 임팩트 보강.

> 핵심: 에셋 완성도에 매달려 본편이 멈추지 않게. **흐름 완성 → 에셋 교체** 순서.
